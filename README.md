# Information Security Lab Projects

A collection of hands-on security projects spanning formal methods, ML security, and  software exploitation + trusted execution environments (TEEs).

## Projects in this repository
This repository contains multiple independent projects under `./projects/`:

- `formal_methods_infosec/` — Protocol modeling + verification (e.g., Tamarin).
- `software_security/` — Security evaluation + attacks against trusted execution environments and research vulnerbilities and exploit them.
- `ml_security/` — Adversarial examples and model privacy attacks.
